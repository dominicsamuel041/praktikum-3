Pengertian Semantic & Non-Semantic Elements

Semantic elements
Semantic elements adalah elemen HTML yang memiliki arti atau makna yang jelas bagi manusia maupun browser. Dengan hanya melihat tag HTML-nya saja, kita bisa mengetahui fungsi elemen tersebut dalam sebuah halaman web.
Non-Semantic elements
Non-semantic elements adalah elemen HTML yang tidak memiliki makna khusus dan hanya berfungsi sebagai wadah untuk elemen lain. Browser tidak dapat memahami tujuan dari elemen ini tanpa adanya informasi tambahan seperti class atau id.
